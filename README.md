# 合一食养（微信小程序）

## 运行方式
1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前仓库：`/Users/fenghe/web-project/heyi-health-ai-mini2`。
4. `AppID` 可先使用测试号或 `touristappid`。
5. 确认 `miniprogramRoot` 为 `miniprogram/`（已在 `project.config.json` 配置）。

- 首页：`pages/home/index`
- 商城：`pages/mall/index`
- 食养：`pages/diet/index`
- AI分析：`pages/analysis/index`
- AI答题：`pages/analysis-quiz/index`
- AI上传：`pages/analysis-upload/index`
